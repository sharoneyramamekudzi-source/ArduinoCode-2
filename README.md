# Arduino Uno Countdown Project

## Overview
This project was developed using an Arduino Uno to demonstrate the use of a 7-segment display and a buzzer. The system performs a countdown from **9 to 0**, displaying each number while producing a short beep before moving to the next number.

## Objectives
- Interface a 7-segment display with an Arduino Uno.
- Generate sound using a buzzer with the `tone()` function.
- Display a countdown sequence from 9 to 0.
- Apply basic Arduino programming concepts such as loops, arrays, and digital output.

## Hardware Used
- Arduino Uno
- 5161AS 7-Segment Display
- Buzzer
- 220 Ω Resistor
- Breadboard
- Jumper Wires
- USB Cable

## Program Features
- Displays numbers from **9** down to **0**.
- Sounds the buzzer during the countdown.
- Automatically repeats the countdown.
